# Sorting Visualizer - a Java software which can visualize some basic sorting algorithms
This software was written in Java, using libGDX framework, it can illustrate some of sorting algoritms:
+ Selection sort
+ Insertion sort
+ Bubble sort
+ Quick sort
+ Counting sort
